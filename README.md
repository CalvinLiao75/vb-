# vb-
使用VB語言製作的躲避球遊戲


遊戲介紹:

在遊戲介面中，地平線會有一個人物，可以左右移動，然後上面會有彈力球，往地板彈，時間越久，球出現越多，
有大有小，人物就要想辦法左右移動閃過這些球，
過程會有箱子空降，箱子裡面會放小驚喜，像是球變小之類的，整個遊戲以時間計時，碰到球即代表遊戲結束。


鑽石模式：每一局會有2條命,地面上會出現鑽石 必須在2分鐘內搜集到20個鑽石 同時躲避球

箱子裡的道具：無敵10秒、延長10秒、鑽石出現數量加倍、加一條命
